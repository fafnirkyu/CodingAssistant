# ML Web Scraper

An enhanced web scraper that combines traditional scraping techniques with machine learning to classify web pages and extract structured data.

## Features

- **ML-Powered Classification**: Classifies web pages into categories using text classification models
- **Structured Data Extraction**: Extracts key information from web pages (titles, links)
- **Text Preprocessing**: Cleans and prepares HTML content for ML processing
- **Extensible Design**: Easy to add new features or improve the classifier
