# Notebooks Directory

This directory contains Jupyter notebooks for experimentation, data exploration, and development.

## Available Notebooks:

### 1. `document_processing.ipynb`
- Exploratory analysis of document structure and content
- Testing different chunking strategies  
- Evaluation of embedding quality metrics

### 2. `embedding_analysis.ipynb` 
- Comparison between various sentence transformer models
- Visualization of embedding spaces using t-SNE
- Performance benchmarking for different configurations

### 3. `qa_evaluation.ipynb`
- Manual evaluation of RAG system responses
- Testing edge cases and query scenarios
- Metrics calculation for answer quality

## Purpose:
These notebooks serve as development tools during the prototyping phase, allowing rapid experimentation with different approaches before implementing them in production code.
